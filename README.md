# Telemaniax
O Telemaniax é um projeto feito para mim unindo duas coisas que eu amo de paixão: Computação e Televisão Brasileira. Um projeto que eu nunca vi antes que transforma o seu terminal linux em um catálogo de canais de TV aberta, desde as mais famosas e as mais importantes a fins de utilidade pública até as mais desconhecidas para os amantes de TV br
